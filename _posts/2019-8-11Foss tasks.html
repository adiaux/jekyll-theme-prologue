Task Description
This task was basically to install Ubuntu and use it as the main operating system to perform the tasks. This was suggested as CUI (Common User Interface)enables programmers to write a single application that can be compiled for each platform.
More importantly it significantly reducing development effort and the command line is a much more direct way to interact.

Task took more time than expected to be completed due to the slow internet connection which made downloading ISO file a long process.
From there everything when pretty smooth.Made a partition of 300GB for it and followed the instructions to complete the task.
